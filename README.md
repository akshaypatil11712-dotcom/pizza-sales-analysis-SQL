# 🍕 Pizza Sales Analysis using SQL

---

## 🎯 Objective
Analyze pizza sales data using SQL to uncover key business insights such as revenue trends, customer preferences, and product performance.

---
## 🔍 Overview

This project focuses on analyzing pizza sales data from a relational database consisting of multiple tables.

Using SQL, various queries were performed to extract insights related to:

* Sales performance
* Customer ordering behavior
* Product demand
* Revenue trends

---

## ❗ Problem Statement

Businesses often struggle to understand:

* Which products perform best
* When peak sales occur
* What drives revenue growth

This project aims to solve these problems using data-driven analysis.

---

## 🎯 Goals

* Identify top-selling pizzas
* Analyze revenue generation
* Understand customer ordering patterns
* Find peak sales hours
* Generate actionable business insights

---

## 📂 Dataset

The dataset consists of 4 relational tables:

### 📌 Orders

* Contains order date and time

### 📌 Order Details

* Contains pizza_id and quantity

### 📌 Pizzas

* Contains size and price

### 📌 Pizza Types

* Contains name, category, and ingredients

---

## 🛠 Tools & Technologies

* 💻 SQL (MySQL)
* 📊 Excel (for visualization - optional)
* 📝 Word (Documentation)

---

## 📊 Key Analysis Performed

### 🔹 Basic Analysis

* Total number of orders
* Total revenue generated
* Highest priced pizza
* Most common pizza size
* Top 5 most ordered pizzas

---

### 🔹 Intermediate Analysis

* Quantity of pizzas by category
* Orders distribution by hour
* Category-wise distribution
* Average pizzas ordered per day
* Top 3 pizzas by revenue

---

### 🔹 Advanced Analysis

* Revenue contribution (%) of each pizza
* Cumulative revenue over time
* Top pizzas per category using ranking

---

## 📈 Key Insights

* Peak orders occur during afternoon hours
* Large-sized pizzas are most popular
* Classic category contributes highest sales
* Few top pizzas generate majority revenue


## 🚀 Business Recommendations

* Increase inventory during peak hours
* Promote low-selling pizzas
* Focus marketing on high-performing categories
* Introduce combo offers to boost sales
